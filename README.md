<p> Esse repositório guarda as aulas e os projetos do Curso de desenvolvimento web completo pela plataforma Udemy. </p>
<p> O curso tem duração de 110 horas, contendo conteúdos e projetos que vão desde as tecnologias Front-end mais básicas, como HTML e CSS, 
até ferramentas para construção de APIs, como Composer e SLIM em PHP.</p>
<p> Ao longo do curso, são desenvolvidos 20 projetos ao todo, cada um referente a uma tecnologia específica ou um conjunto delas.</p> <br>

## Lista de tecnologias e ferramentas ensinadas no curso
<ul>
  <li> HTML5 </li> 
  <li> CSS3 </li>
  <li> Bootstrap 4 & Design responsivo </li>
  <li> JavaScript </li>
  <li> JavaScript Moderno (ECMAScript 2015 - ES6) </li>
  <li> PHP 7 e Orientação a Objetos </li>
  <li> Banco de dados MySQL </li>
  <li> PHP com PDO </li>
  <li> JQuery </li>
  <li> API com Slim Framework </li>
  <li> SASS </li>
  <li> IONIC </li>
</ul> <br>
